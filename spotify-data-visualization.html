Spotify Data Visualization

Interactive visualizations exploring user listening habits and playlist dynamics.

Role

Visual Designer

Category

Data Visualization / Interactive Design / User Experience

Artifacts

05

Status

Selected Work

Challenge

How might music data be transformed into an interactive visual experience that communicates both information and emotion while remaining intuitive to explore?

Solution

I designed a modular visualization instrument that converts audio datasets into responsive visual forms. Using a bento-style interface, users can manipulate controls such as sliders and rotary knobs to influence how data is represented in real time.

Outcome

The resulting prototype reimagines data exploration as an active and creative process, combining principles of information design, interaction design, and generative visualization into a unified system.
